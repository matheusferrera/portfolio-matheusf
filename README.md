Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para apresentar minha trajetória profissional, projetos e habilidades técnicas como Desenvolvedor Full-stack.

---

## ✨ Destaques

- 🌍 **Internacionalização (i18n)**: Suporte completo para Português, Inglês e Espanhol.
- 🌓 **Modo Escuro/Claro**: Interface adaptável com toggle de tema.
- 🚀 **Performance Extrema**: Construído com **Astro 6**, garantindo carregamento instantâneo e SEO otimizado.
- 🎨 **Design Moderno**: Estilização com **Tailwind CSS 4** e animações suaves via **Motion**.
- 🖱️ **Smooth Scroll**: Experiência de navegação fluida utilizando **Lenis**.

---

## 🛠️ Tecnologias Utilizadas

### Core
- **Framework:** [Astro 6](https://astro.build/)
- **Linguagem:** TypeScript
- **Estilização:** Tailwind CSS 4

### Bibliotecas & Ferramentas
- **Animações:** [Motion (framer-motion)](https://motion.dev/)
- **Scroll Suave:** [Lenis](https://lenis.darkroom.engineering/)
- **Ícones:** Lucide Icons (através de componentes Astro)

## 📂 Estrutura do Projeto

```text
/
├── public/          # Ativos estáticos (imagens, ícones)
├── src/
│   ├── components/  # Componentes reutilizáveis (Header, Experience, Projects, etc.)
│   ├── i18n/        # Configurações e dicionários de tradução
│   ├── layouts/     # Layouts principais da aplicação
│   ├── pages/       # Rotas e páginas (index.astro)
│   └── styles/      # Arquivos CSS globais
└── astro.config.mjs # Configurações do Astro e Integrações
```

## ⚙️ Como Executar Localmente

Siga os passos abaixo para rodar o projeto em sua máquina:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/projeto-portfolio.git
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Acesse no navegador:**
   Abra [http://localhost:4321](http://localhost:4321) para visualizar o resultado.
